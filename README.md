# todo-react

## Description
This project is a **group project** developed as part of a collaborative assignment.  
It is a web application built using **React and Vite**, designed to provide a fast, modern, and responsive user experience.

---

## Technologies Used
- React
- Vite
- JavaScript
- HTML
- CSS
- Supabase
- ESLint

---

## Features
- Responsive user interface
- Fast development environment with Vite
- Component-based React structure
- Backend/database integration using Supabase
- Organized project structure
- Reusable components and scripts

---

## How to Run
Open it directly in **VS Code** or your development environment and run the Vite development server.

---

## Output
- Functional React web application
- Dynamic frontend pages
- Database-connected features
- Responsive design

---

## Notes
- This project was developed as a **team/group project**
- Uses Vite for fast build and hot reload
- Supabase is used for backend/database functionality
